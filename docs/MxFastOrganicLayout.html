MxFastOrganicLayout

MxFastOrganicLayout()

Extends to implement a fast organic layout algorithm. The vertices need to be connected for this layout to work, vertices with no connections are ignored. Example: ```javascript let layout = new mxFastOrganicLayout(graph); layout.execute(graph.getDefaultParent()); ``` Constructor: mxCompactTreeLayout Constructs a new fast organic layout for the specified graph.
