Thank you for your interest in contributing to maxGraph! We welcome contributions from the community and appreciate your help in making this project better.

For comprehensive information about contributing, including:
- Communication requirements before starting work
- Issue reporting guidelines and templates
- Development setup and workflow

- Pull request process

Please see our **[Contributing Guide](https://github.com/maxGraph/maxGraph/blob/main/CONTRIBUTING.md)** in the repository.
